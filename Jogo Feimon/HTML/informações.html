Sobre o Feimon Farm

Feimon Farm é um jogo inspirado principalmente em Pokemon e uma tentativa de Stardew Valley, jogos ao qual aprecio muito

Feimon é um jogo aonde é totalmente sobre batalha de turnos e acumulo de pontos a cada batalha vencida com um chefe no final

Tecnologias Usada

  • HTML
  • JavaScript
  • CSS

Tutorias,Tecnologias e Materias de terceiros utilizados para criação

  • Videos genericos sobre HTML e JavaScript Para melhor entedimento de animação e canvas
